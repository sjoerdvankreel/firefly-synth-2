# High level overview
* Microtuning support by means of MTS-ESP
* Per-voice realtime visuals for graphs and knobs
* Some read-to-use modulators without having to set up the matrix
* Detailed audio routing options without having to set up a matrix

# 4x Oscillator
* Various basic waveforms
* Dedicated FM oscillator
* Combined string/noise algorithm
* Inter-oscillator AM/FM options
* Allows external audio as oscillator
* Optional oversampling

# 4x Per-voice and 4x global effects
* Comb filter
* State variable filter
* Various waveshapers
* Optional oversampling

# Per-voice and global echo
* Reverb
* Feedback delay
* Multitap delay
* Built-in filters

# 6x Per-voice and 6x global LFO
* 3 subslots for each
* Various waveforms
* Skewing options for the first subslot

# 7x Per-voice envelope
* 16-stage MSEG with visual editor
* Customizeable loop and release points

# Extensive matrix module
* 25 per-voice and 12 global slots

# Extensive global unison module
* 38 targetable per-voice parameters
* Optional full manual control over per-voice parameter positioning
